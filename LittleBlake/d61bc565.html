前言

静态博客系统不像 Wordpress 一样自带评论系统，本文将基于Hexo Butterfly 主题介绍 Giscus 评论系统添加方法。

Giscus

giscus 是一个基于GitHub Discussions 的评论插件。 它的特点是简单易用，无需注册，支持多种主题，支持多语言，支持自定义配置。 所有评论都存储在你的GitHub Discussions 中，不会丢失，方便管理。

具体步骤

1. 创建github仓库

创建一个公开的 GitHub 仓库，可以叫做 blog-comments

2. 开启仓库的Discussions

点击仓库的settings

在General中向下滚动，找到Features部分，勾选开启Discussions

3. 安装Giscus App

访问 https://github.com/apps/giscus 点击 Install，进入选择仓库界面
选择 Only select repositories ，然后选择刚才创建的 blog-comments 仓库，点击 Install

4. 获取Giscus配置

访问 Giscus官网

  • 选择语言
  • 设置仓库，格式：你的用户名/blog-comments
  • 映射关系：默认
  • Discussion分类：Announcements
  • 特性：默认
  • 主题：默认或自选
